# PWA Icons

Place the following PNG files here for full PWA support:

- `icon-192x192.png` — Android homescreen icon
- `icon-512x512.png` — Splash screen / high-res icon

Recommended: Use your logo on a black (#0A0A0A) background with beige (#E8D5B7) art.
Tools: https://realfavicongenerator.net or https://maskable.app
